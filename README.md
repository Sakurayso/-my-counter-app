# ✨ 互动计数器 & 灵感画廊
一个现代化、高颜值的纯前端互动网页项目，结合 **点击计数功能** 与 **随机美图画廊**，采用玻璃拟态设计 + 动态渐变背景，流畅美观、开箱即用。

---

## 📖 项目介绍
本项目是一个纯前端实现的互动式网页，无需后端、无需配置环境，直接在浏览器运行。
集成交互计数、数据本地存储、随机图片切换、动态UI动画等多种现代化前端特性。

适合：
- 前端入门学习
- 课程设计/大作业
- 个人Demo展示
- 界面设计参考

---

## 🌟 功能特性
✅ **精美UI设计**
- 玻璃拟态（Glassmorphism）风格
- 动态流动渐变背景
- 柔和渐变配色 + 卡片阴影

✅ **互动计数器**
- 点击按钮数字自增
- 数字弹跳动画效果
- 计数本地持久化（刷新不丢失）

✅ **随机美图画廊**
- 每次重置自动切换新图片
- 图片平滑加载动画
- 响应式图片展示

✅ **全端适配**
- 支持电脑、平板、手机自适应
- 按钮悬停/点击动效
- 操作流畅无卡顿

✅ **纯前端运行**
- 单HTML文件，开箱即用
- 无需服务器、无需依赖包
- 本地直接打开运行

## 🛠️ 技术栈
- **HTML5** — 页面结构
- **CSS3** — 样式、渐变、动画、Flex 布局
- **JavaScript** — 交互逻辑、本地存储
- **LocalStorage** — 计数数据持久化
- **Glassmorphism** — 毛玻璃视觉风格

## 🚀 快速使用
### 1. 下载文件
将代码保存为：`index.html`

### 2. 直接运行
**双击文件** → 使用任意浏览器打开即可运行。

### 3. 功能操作
1. 点击 **🚀 点击增加能量**：计数+1，播放动画
2. 点击 **🔄 重置并换图**：计数归零，自动更换新图片
3. 关闭页面重新打开：计数自动恢复
